vdom

vdom.userSettings(value[, loadName, saveName])

Applies object values to Application property userSettings. (ESM only)

Parameters:

  • value (UserSettings) – PlainObject with settings overrides

  • loadName (string) – Restore previously saved settings from local storage

  • saveName (string) – Store current settings into local storage

Usage:

import { userSettings } from "squared/vdom.js";

userSettings({
  pierceShadowRoot: true,
  adaptStyleMap: true
});
